# Campus Recruitment Automation System (CRAS)

# Overview

Campus Recruitment Automation System (CRAS) is a web-based platform developed to automate and simplify the campus placement process for students, recruiters, and placement officers.

The system provides a centralized platform for job posting, student registration, resume generation, application tracking, interview scheduling, and placement management.

# Student Module

* Student Registration and Login
* Profile Management
* Resume Generation
* View Available Job Openings
* Apply for Jobs
* Track Application Status

# Admin Module

* Manage Students and Recruiters
* Post Job Opportunities
* Schedule Interviews
* Send Notifications
* Generate Placement Reports

# Recruiter Module

* Post Job Requirements
* View Student Applications
* Access Standardized Resumes
* Manage Recruitment Process
